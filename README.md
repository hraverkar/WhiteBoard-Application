# WhiteBoard Angular Application 
  Angular 9 boiler plate with footer, header and material module enabled.

## Tech Stack

* Angular 9
* Material Module
* Node.js
* Konva

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
